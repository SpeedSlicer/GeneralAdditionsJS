API

Cookie Management

setCookie(name, value, days): Sets a cookie.
getCookie(name): Retrieves the value of a cookie.
deleteCookie(name): Deletes a cookie.
clearAllCookies(): Clears all cookies.

Local Storage and Session Storage

setLocalStorageWithExpiration(key, value, expirationMins): Stores data in local storage with expiration.
getLocalStorageWithExpiration(key): Retrieves data from local storage if not expired.
setSessionStorage(key, value): Stores data in session storage.
getSessionStorage(key): Retrieves data from session storage.

Date and Time

formatDate(date): Formats a date object into "YYYY-MM-DD HH:MM".
getCurrentUTCTimestamp(): Retrieves the current UTC timestamp in milliseconds.

Random Data Generation

generateRandomString(length): Generates a random alphanumeric string of a specified length.
generateUUID(): Generates a universally unique identifier (UUID).

Data Conversion and Manipulation

base64ToBlob(base64, contentType): Converts a base64 string to a Blob object.
bytesToSize(bytes): Converts bytes to a human-readable file size string.

Utility and Validation

copyToClipboard(text): Copies a string to the clipboard.
isValidEmail(email): Validates if a string is a valid email address.
isValidURL(url): Validates if a string is a valid URL.

DOM Manipulation and Utilities

createAbout(url, title, faviconURL): Creates a new window with an iframe displaying a URL.
scrollToTop(): Scrolls the page to the top.
scrollToElement(selector): Scrolls the page to a specific element.

Browser Detection and Feature Support

isMobileDevice(): Detects if the device is a mobile device.
isInternetExplorer(): Detects if the browser is Internet Explorer.
supportsIntersectionObserver(): Checks if the browser supports Intersection Observer API.

String Manipulation

capitalizeFirstLetter(str): Capitalizes the first letter of a string.
toTitleCase(str): Converts a string to title case.

Async Operations and Promises

fetchData(url, options): Fetches data from a URL using Fetch API with error handling.
delay(ms): Returns a promise that resolves after a specified delay.

UI and Interface Handling

setDocumentTitle(title): Sets the title of the current document.

Miscellaneous

getRandomInt(min, max): Generates a random integer between specified ranges.
